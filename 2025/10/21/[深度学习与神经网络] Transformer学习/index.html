注意力机制计算公式

缩放点积注意力（Scaled Dot-Product Attention）

Attention(Q,K,V)=softmax ⁣(QKTdk)V\text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^{T}}{\sqrt{d_k}}\right) V Attention(Q,K,V)=softmax(dk​​QKT​)V

解释：

  1. QKTQK^TQKT 表示 Query 与 Key 的相似度。
  2. 除以 dk\sqrt{d_k}dk​​ 是为了防止点积结果过大导致梯度消失。
  3. softmax 将相似度转化为概率分布。
  4. 与 VVV 相乘后得到加权求和的结果，即注意力输出。

多头注意力（Multi-Head Attention）

MultiHead(Q,K,V)=Concat(head1,…,headh)WO\text{MultiHead}(Q,K,V) = \text{Concat}(\text{head}_1, \dots, \text{head}_h)W^O MultiHead(Q,K,V)=Concat(head1​,…,headh​)WO

其中每个注意力头计算为：

headi=Attention(QWiQ,KWiK,VWiV)\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V) headi​=Attention(QWiQ​,KWiK​,VWiV​)

公式关系总结：

  • 每个头的维度：dk=dv=dmodelhd_k = d_v = \frac{d_{\text{model}}}{h}dk​=dv​=hdmodel​​
  • 最终输出维度仍为 dmodeld_{\text{model}}dmodel​

核心公式

Attention = softmax(QKTdmodel)V\text{softmax}\left(\frac{QK^{T}}{\sqrt{d_{\text{model}}}}\right)Vsoftmax(dmodel​​QKT​)V
